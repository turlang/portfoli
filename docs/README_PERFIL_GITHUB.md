**Tecnologias:** React, Vite, Node.js, Fastify, TypeScript, Prisma, MongoDB e JWT.

### LeadHunter Pro

SaaS de prospecção comercial com geração de leads, CRM, pipeline, exportação CSV, planos de assinatura, Mercado Pago, painel administrativo e segurança anti-abuso.

**Tecnologias:** Node.js, Express, MongoDB, Mongoose, JWT, Mercado Pago, HTML, CSS e JavaScript.

---

### SaaS Mestre Orc

Plataforma para mestres de RPG criarem e gerenciarem campanhas, personagens, reservas, conteúdos gerados e exportações para ecossistemas de RPG digital.

**Tecnologias:** JavaScript, dashboards, simulação SaaS, geração de conteúdo, exportação JSON e arquitetura de produto.

### Papel Parede

Landing page comercial em React para venda de papéis de parede, com catálogo, filtros, oferta, CTA para WhatsApp e visual responsivo.

**Tecnologias:** React, Vite, JavaScript, CSS e UX/UI.

### Orçamento Pessoal

Dashboard financeiro com PWA para controle de receitas, despesas, metas e previsão mensal.

**Tecnologias:** HTML, CSS, JavaScript, Service Worker e PWA.
