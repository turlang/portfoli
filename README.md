# Portfolio Profissional v11

Site pessoal profissional de Evandro Ricardo de Souza.

## Como testar
Abra `index.html` no navegador.

Opcional com servidor local:

```bash
npx serve .
```

## Estrutura
- `index.html`: página inicial
- `sobre.html`: perfil acadêmico/profissional
- `projetos.html`: projetos em formato de estudos de caso
- `assets/css/styles.css`: estilos
- `assets/js/main.js`: menu e carrossel 3D

Sem dependências obrigatórias.
